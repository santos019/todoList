# todoList
### 프로젝트 목적
> 할 일 목록을 관리하는 어플리케이션 개발

### 개발 환경
> + JavaScript  
> + webpack: ^5.73.0  
> + eslint: ^8.15.0

### 실행 메뉴얼
> + #### 개발 모드  
>> ``` npm run serve  ```
> + #### build  
>> ``` npm run build ```  
> => dist 폴더에 build된 파일들이 생성
> + #### build 파일 실행  
>> ``` cd dist ```  
>> ``` npm run serve ```
